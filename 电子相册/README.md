# 3D 爱心星空相册

直接双击 `index.html` 就能打开预览。页面底部的“选择照片”可以临时选你电脑里的照片看效果。

现在已经放入 21 张照片，并把顶部标识改成 `laq`。

交互方式：

1. 鼠标滚轮可以控制相册进度，往下滚前进，往上滚后退。
2. 鼠标悬停在照片上会轻微放大。
3. 按住照片可以拖出来，不会影响整体排版，进入下一段排版时会自动回去。
4. 点击任意照片会打开大图、仿视频弹窗文案，并可以写留言备注。
5. 保存留言后，备注会存到浏览器本地，刷新页面也还在。

如果你以后想自己改照片，可以这样做：

1. 把图片复制到 `photos` 文件夹。
2. 打开 `photos.js`。
3. 按这个格式填写：

```js
window.ALBUM_PHOTOS = [
  "./photos/001.jpg",
  "./photos/002.jpg",
  "./photos/003.jpg"
];
```

建议准备 20 到 35 张照片，竖图效果最好。
